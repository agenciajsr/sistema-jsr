  <action>
    Criar `src/actions/clientes.ts` (`'use server'`) com 3 exports:

    1. `createClienteComContrato(clienteInput: ClienteInput, contratoInput: ContratoInput)`:
       - Valida `clienteInput` com `clienteSchema.safeParse` e `contratoInput` com `contratoSchema.safeParse`; se algum falhar, retorna `{ error: 'Não foi possível salvar. Verifique os dados e tente novamente.' }` (copy exata do UI-SPEC).
       - Usa `db.transaction(async (tx) => { ... })`: insere o cliente em `clientes`, obtém o `id` gerado, chama `construirRegistroRenovacao(clienteId, { dataInicio: contratoInput.dataInicio, dataVencimento: contratoInput.dataVencimento, valorMensal: contratoInput.valorMensal })`, insere o resultado em `contratos` via `tx.insert(contratos).values(registro)`.
       - Retorna `{ data: { clienteId } }` em sucesso.

    2. `updateCliente(id: string, input: ClienteInput)`:
       - Requer usuário autenticado (`getCurrentUser()`; se `null`, retorna `{ error: 'Sessão expirada. Faça login novamente.' }`).
       - Valida `input` com `clienteSchema`.
       - `db.update(clientes).set({ ...input, updatedAt: new Date() }).where(eq(clientes.id, id))`.
       - Retorna `{ data: { id } }`.

    3. `deleteCliente(id: string)`:
       - Chama `requireAdmin()` PRIMEIRO — se retornar `{ error }`, devolve esse erro imediatamente (D-03: exclusão é exclusiva do Admin). Mensagem: "Apenas administradores podem excluir clientes."
       - Se Admin, `db.delete(clientes).where(eq(clientes.id, id))` (o `onDelete: 'cascade'` do schema remove os contratos vinculados automaticamente).
       - Retorna `{ data: { id } }`.
  </action>

       - Chama `construirRegistroRenovacao(clienteId, { dataInicio: input.dataInicio, dataVencimento: input.dataVencimento, valorMensal: input.valorMensal })`.

       - Chama `requireAdmin()` PRIMEIRO — se retornar `{ error }`, devolve imediatamente. Mensagem: "Apenas administradores podem excluir contratos."

       - Retorna `{ data: { id } }`.

    3. `getContratosDoCliente(clienteId: string)`:
       - `db.query.contratos.findMany({ where: eq(contratos.clienteId, clienteId), orderBy: [desc(contratos.dataInicio)] })`.
       - Aplica `selecionarContratoAtual(lista)` sobre o resultado para derivar o contrato vigente.
       - Retorna `{ contratoAtual, historico: lista }` (onde `historico` é a lista completa, incluindo o atual, para exibição na seção "Histórico de Contratos" do Plan 01-08).
